# Ayoub Mallah — Win95 Portfolio

A static, Windows 95-themed personal site. No build step, no dependencies — just HTML/CSS/JS.

## Structure
- `index.html` — desktop, icons, and all window content (About, Resume, Projects, Contact, Recycle Bin)
- `style.css` — all Win95 styling (bevels, title bars, taskbar, start menu)
- `script.js` — window open/close/minimize/maximize, taskbar, start menu, clock

## Notes
- Fully responsive down to mobile (icons open on single tap instead of double-click when a touch screen is detected)
- Font falls back to system sans-serif since real "MS Sans Serif" isn't a web-safe font.
